# cisco-webapp-api

## Instructions

Go to https://www.reddit.com/prefs/apps and get a new client ID and client secret. Then run

```
export CWA_ID=your_client_id
export CWA_SECRET=your_client_secret

pip install -r requirements.txt
flask run
```
